# Hangman Python

A fun command-line Hangman game built using Python.

## Features

- Random word selection
- Lives system
- Letter validation
- Win/Loss detection
- Replay support

## Technologies Used

- Python 3
- Random Module

## Installation

```bash
git clone https://github.com/yourusername/hangman-python.git
cd hangman-python
python hangman.py
```

## How to Play

1. A random word is selected.
2. Guess one letter at a time.
3. Correct letters are revealed.
4. Wrong guesses reduce your lives.
5. Guess the word before lives reach zero.

## Example

```text
Word: _ _ _ _ _

Guess a letter: a

Correct!

Word: A _ _ _ _
Lives Left: 6
```
